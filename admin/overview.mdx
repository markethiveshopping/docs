The MarketHive admin dashboard is your command center for keeping the platform running smoothly. From here, you oversee every job listing, user account, payment transaction, and review on the platform — giving you the tools to enforce quality standards, support users, and monitor platform health in real time. Whether you're approving new listings, investigating a user complaint, or tracking today's revenue, the admin dashboard puts everything you need within reach.

## Accessing the Admin Dashboard

To log in to the admin dashboard, navigate to `/admin` and enter your admin credentials. The dashboard is restricted to authorized personnel only.

<Note>
  Admin access is granted exclusively by the platform owner. If you need access or believe your permissions are incorrect, contact the platform operator directly to request the appropriate role.
</Note>

## Dashboard Home

When you first log in, the dashboard home screen gives you an at-a-glance snapshot of platform activity. The summary stat cards at the top of the screen display the following metrics:

| Metric | Description |
|---|---|
| **Total Listings** | The cumulative number of job listings ever posted on the platform |
| **Active Users** | The number of user accounts currently active and in good standing |
| **Today's Unlocks** | The number of contact-detail unlocks purchased by job seekers today |
| **Total Revenue** | Cumulative platform revenue from all unlock transactions |
| **Pending Reviews** | The number of user-submitted reviews awaiting moderation |

Below the summary cards, you'll find a live activity feed showing recent listings, new user registrations, and flagged content that requires your attention.

## Key Admin Responsibilities

As a MarketHive admin, your core responsibilities span four areas:

**Listing Quality** — You are the final gate before a job listing goes live. Review every new listing in the moderation queue for accuracy, relevance, and compliance with platform policy before approving it.

**User Integrity** — You manage the accounts of both employers and job seekers. This includes verifying legitimate employers, investigating reports of misconduct, and suspending or banning accounts that violate the platform's terms of service.

**Payment Oversight** — Job seekers pay a one-time fee to unlock the contact details on each listing. You have visibility into all transactions and can investigate disputed payments or unlock issues through the Payments section.

**Review Moderation** — Reviews build trust on the platform. You approve, reject, and remove reviews to ensure that only genuine, policy-compliant feedback appears on listings.

<Tip>
  Start each session by checking the dashboard home for pending listings, pending reviews, and any new user reports flagged overnight. Clearing the moderation queue regularly ensures employers and job seekers have a responsive experience.
</Tip>
